# HTML & CSS Notes - Beginner Folder 📘

Ye sab files HTML aur CSS ke alag-alag topics par hain. Har file ko **browser mein khol ke dekho** — code ke saath comments diye hain jo line-by-line samjhate hain ki kya ho raha hai.

## Padhne ka order (sequence):

### HTML Basics
1. **01-headings-paragraphs.html** — Headings (h1-h6), paragraphs, `<br>`, `<hr>`
2. **02-lists.html** — Unordered list (ul), Ordered list (ol), Description list (dl)
3. **03-links-images.html** — Links (`<a>` tag) aur Images (`<img>` tag)
4. **04-tables.html** — Tables, colspan/rowspan, thead/tbody/tfoot
5. **05-forms.html** — Forms, input types, radio, checkbox, dropdown, textarea
6. **06-text-formatting.html** — Bold, italic, underline, strikethrough, highlight
7. **07-div-span-semantic.html** — div, span, aur semantic tags (header, nav, main, etc.)

### Practice Files (Apne khud likhe hue)
8. **08-Form.html** — Forms tag basic details
9. **09-animal.html** — `animal.css` use karke image ko box me set karna aur animal breed type seekhna
10. **10-website practice.html** — Pehli website design ("DEVNova" naam se) HTML + CSS use karke
11. **11-Form.html** — Form ke advance tags use karke
12. **12-form.html** — Form ko doosri style me design karna
13. **13-website2 practice.html** — Practice ke liye doosri website design

### CSS Notes
14. **14-css-basics.html** — CSS kya hai, 3 tarike CSS lagane ke, Internal CSS, selectors (tag/class/id), common properties **15-basic-css.html** + **-basic-css.css** — External CSS file kaise link karte hain, basic selectors aur box model (in dono files ko SAME folder me rakhna)
15. **15-advanced-css.html** + **advanced-css.css** — Flexbox, Grid, Positioning, Transitions, Pseudo-classes (in dono files ko bhi SAME folder me rakhna)
16. **16-css-deep-dive.html** + **css-deep-dive.css** — Background (color/image/gradient/shorthand), Colors (HEX/RGB/RGBA/HSL), Units (px/%/em/rem/vh/vw), Display (block/inline/inline-block/none), Float, Box-shadow, Text-shadow (in dono files ko bhi SAME folder me rakhna)

## Kaise use karein:

- Har file ko double-click karke browser mein kholo (Chrome/Firefox kisi mein bhi)
- Code ko text editor mein bhi kholo (VS Code recommended) taaki comments padh sako
- HTML files me comments: `<!-- ye notes hain -->`
- CSS files me comments: `/* ye notes hain */`
- Jin files ke 2 parts hain (HTML + CSS), unhe hamesha SAME FOLDER me rakhna — warna styling load nahi hogi
- Browser mein result dekho, fir code padho, fir comments padho — teeno match karke samjho

## Tip junior developers ke liye:

Sirf padhna kaafi nahi hai — file ko copy karo, values change karo (jaise apna naam, apni list ke items), aur dekho result kaise change hota hai. Yahi sabse best tarika hai seekhne ka.
