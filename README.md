# HTML Notes - Beginner Folder 📘

Ye sab files HTML ke alag-alag topics par hain. Har file ko **browser mein khol ke dekho** — code ke saath comments diye hain jo line-by-line samjhate hain ki kya ho raha hai.

## Padhne ka order (sequence):

1. **01-headings-paragraphs.html** — Headings (h1-h6), paragraphs, `<br>`, `<hr>`
2. **02-lists.html** — Unordered list (ul), Ordered list (ol), Description list (dl)
3. **03-links-images.html** — Links (`<a>` tag) aur Images (`<img>` tag)
4. **04-tables.html** — Tables, colspan/rowspan, thead/tbody/tfoot
5. **05-forms.html** — Forms, input types, radio, checkbox, dropdown, textarea
6. **06-text-formatting.html** — Bold, italic, underline, strikethrough, highlight
7. **07-div-span-semantic.html** — div, span, aur semantic tags (header, nav, main, etc.)
8. **08-Form.html** - forms tag basic details
9. **09-animal.html** -using animal.css code  learn to set image in box and make animal breed type
10. **10-website practice.html** -design my first website using html and css nam DEVNova
11. **11-Form.html**-using advance tags in form
12.  **12-form.html** design  form in other style
13. **13-website2 practice.html** -design some other website for practice

## Kaise use karein:

- Har file ko double-click karke browser mein kholo (Chrome/Firefox kisi mein bhi)
- Code ko text editor mein bhi kholo (VS Code recommended) taaki comments padh sako
- Comments format: `<!-- ye notes hain -->` — yahi tumhare notes hain, code ke andar hi
- Browser mein result dekho, fir code padho, fir comments padho — teeno match karke samjho

## Tip junior developers ke liye:

Sirf padhna kaafi nahi hai — file ko copy karo, values change karo (jaise apna naam, apni list ke items), aur dekho result kaise change hota hai. Yahi sabse best tarika hai seekhne ka.
